JavaScript: JavaScript is a programming language commonly used in web development.

Operations: Actions that can be performed in programming.

Variables: Containers for values or strings.

Variable naming conventions: Camel Case

Functions: A block of code that performs an action and returns a result

Methods: an action run on a piece of data

Arguments: The data you pass to your function

Parameters: The names listed in the function definition

Return: Stops the execution of a function and returns a value from that function

Chaining methods: allows you to chain methods together.

Strings: A Data type that is a collection of characters in quotations.

Booleans: A data type that represents true and false values.

undefined: A data type that indicates the container is empty.

NaN: 'Not A Number'

Escape: Using a backslash to indicate that a character is to be read literally.

Alert: A popup that sends a message to the user.

Comments: Text in code that is used to give reminders/notes. Not actual code.

jQuery: A popular Javascript library that adds front end interactivity.

Attribute: Text used to change the nature of HTML elements.
